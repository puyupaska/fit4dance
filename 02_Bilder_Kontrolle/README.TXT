==========================
mediDOK PACS Patienten-CD
==========================
Stand der Info-Datei: 19. Feb 2016 / Viewer-Version 2.0 R7 SP3


Sehr geehrte Damen und Herren,

diese CD oder DVD wurde mit dem mediDOK PACS System erstellt und
enthält Bilddaten Ihrer Untersuchung. Die Bilddaten können mit dem 
ebenfalls auf der DC/DVD vorhandenen Viewer betrachtet werden.

Starten des mediDOK Bildviewers (DICOM):
----------------------------------------

- Wenn Ihr CD/DVD-Laufwerk einen AutoStart unterstützt, startet der Viewer
  automatisch nach Einlegen der CD.
- Um den Viewer manuell zu starten, führen Sie bitte die Datei MEDIDOK.EXE 
  auf der CD/DVD aus.

Systemvoraussetzungen:
----------------------

Der vorhandene Viewer kann auf Windows-Systemen (Windows XP oder höher) 
verwendet werden und benötigt keine Installation.


Zugriff auf Bilddaten (ohne Viewer):
------------------------------------

Das Medium enthält die Bilddaten im DICOM-Format. Diese können von
DICOM-kompatiblen Viewern gelesen werden. Dafür sind folgende Dateien
auf dem MEDIUM relevant:

- Datei "DICOMDIR" --> eine Inhaltsdatei, die von DICOM-Viewern i.d.R.
  gelesen werden kann.

- Verzeichnis "IMAGES" --> dies sind die einzelnen DICOM-Bilder.

Zusätzlich (je nach Einstellung auf dem System des Erstellers der CD/DVD)
finden Sie die Bilddaten auch im JPG-Format in folgenden Ordner: 

- Ordner "JPEG". 

Diese Bilddaten können i.d.R. mit jedem gängigen Bildbetrachter geöffnet 
werden.

Eine HTML-Seite mit direktem Zugriff auf die JPG-Bilddaten finden Sie hier:

- Datei "HTML\index.htm"


Hilfe:
------

Bei Problemen mit der CD/DVD wenden Sie sich bitte zunächst an die Praxis oder
Klinik, die das Medium erstellt hat.
